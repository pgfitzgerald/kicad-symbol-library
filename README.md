# Lucian Audio's KiCad Symbol Library

![Lucian Audio's KiCad Symbol Library](https://pgfitzgerald.github.io/kicad-symbol-library/images/hero-banner-1k.jpg)

## About

This is a KiCad 10.0 symbol library containing schematic symbols for vacuum tubes.

## Contents

- **JJ Electronic Vacuum Tubes**

## Installation

1. Download [kicad-symbol-library.master.zip](https://github.com/pgfitzgerald/kicad-symbol-library/archive/refs/heads/master.zip)
2. Extract the archive
3. Copy the desired `.kicad_symdir` directories to a location of your choice
   - Windows Example: %USERPROFILE%\Documents\KiCad\10.0\symbols
   - macOS/Linux Example: ~/Documents/KiCad/10.0/symbols
4. Follow the instructions within the [KiCad Docs](https://docs.kicad.org/10.0/en/eeschema/eeschema.html#managing-symbol-libraries) to add the symbol libraries to your preferred symbol library table

## Features

- Symbols adhere to [KiCad Library Conventions](https://klc.kicad.org/) as closely as possible
- Thoughtful design patterns
- Consistent visual aesthetics
- Accurate component details
